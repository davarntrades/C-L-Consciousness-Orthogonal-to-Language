## The Five Rules of Compression

```
════════════════════════════════════════════════════════════════

  RULE 1:
  ─────────────────────────────────────────────────────────────
  Remove everything that doesn't change the outcome.

  If removing it does not alter the behaviour of the system —
  it is noise.
  Not depth. Not nuance. Not sophistication.
  Noise.

  Most academic writing is noise
  dressed as rigour.
  Most corporate communication is noise
  dressed as strategy.
  Most philosophy is noise
  dressed as profundity.

  Remove it.
  If the system behaves the same without it:
  it was never part of the idea.

  ─────────────────────────────────────────────────────────────

  RULE 2:
  ─────────────────────────────────────────────────────────────
  Identify the invariant.

  The thing that stays true
  no matter how you transform the system.
  No matter the substrate.
  No matter the scale.
  No matter the context.

  If it only holds sometimes:
  not an invariant.
  Keep looking.

  The invariant is what you are after.
  Everything else is commentary.

  ─────────────────────────────────────────────────────────────

  RULE 3:
  ─────────────────────────────────────────────────────────────
  Express it in the fewest symbols possible.

  If it takes 20 paragraphs:
  you do not understand it yet.

  Understanding is compression.
  Not explanation.

  You understand something
  when you can state it
  in fewer symbols than it took
  to discover it.

  Einstein understood gravity
  when he compressed it to curvature.
  Before that: equations filling pages.
  After: geometry. One word.

  ─────────────────────────────────────────────────────────────

  RULE 4:
  ─────────────────────────────────────────────────────────────
  Language is the last step. Not the first.

  Most people start with language.
  They write to think.
  They use words to find the idea.
  The idea arrives, if it arrives,
  somewhere in paragraph seven.
  Surrounded by the scaffolding
  that was used to build it.
  The scaffolding stays.
  The idea is buried inside it.

  Structure first.
  Language last.

  Think in structures.
  When the structure is correct —
  the language writes itself.
  In the fewest words.
  Because it has nowhere else to go.

  ─────────────────────────────────────────────────────────────

  RULE 5:
  ─────────────────────────────────────────────────────────────
  Compression is the test of truth.

  If an idea fully compresses: it is real.
  If it collapses under compression: it was fluff.

  This is not harsh.
  This is the most useful test available.

  Apply it to everything.
  Apply it to this document.
  Apply it to your own thinking.
  Whatever survives compression
  is what is actually there.
  Whatever disappears under compression
  was never there.
```

## What It Means to Think in Structures

```
════════════════════════════════════════════════════════════════

  THINKING IN SENTENCES:

  Idea arrives as language.
  Words suggest other words.
  Paragraphs suggest paragraphs.
  The thinking happens in the writing.
  The structure, if it forms,
  forms accidentally.
  From the arrangement of sentences.

  Language is load-bearing.
  Remove the language:
  nothing remains.

  ─────────────────────────────────────────────────────────────

  THINKING IN STRUCTURES:

  Idea arrives as geometry.
  A relationship between things.
  A constraint. An invariant. A shape.
  The language comes later —
  to describe what was already seen.

  Language is decorative.
  Remove the language:
  the structure remains.
  It was always there.
  Independent of its description.

  ─────────────────────────────────────────────────────────────

  DIAGRAM:

  ┌─────────────────────────────────────────────────────────┐
  │                                                         │
  │   SENTENCE THINKER:                                     │
  │                                                         │
  │   Words → Words → Words → Words → [idea maybe]         │
  │                                                         │
  │   If you remove the words: nothing.                     │
  │   The words were the thinking.                          │
  │                                                         │
  │   STRUCTURE THINKER:                                    │
  │                                                         │
  │   Structure → [seen completely] → words (optional)     │
  │                                                         │
  │   If you remove the words: structure remains.           │
  │   The structure was the thinking.                       │
  │   The words were the translation.                       │
  │                                                         │
  └─────────────────────────────────────────────────────────┘

  This is why compression is natural
  for structure thinkers.

  They are not compressing language.
  They never inflated it in the first place.

  They are translating structure into symbols.
  The translation is short
  because the structure is precise.

  Precision compresses.
  Imprecision expands.
```

## The Compression Hierarchy

```
════════════════════════════════════════════════════════════════

  LEVEL 0 — No understanding:
  Cannot explain it at all.

  LEVEL 1 — Surface understanding:
  Can repeat what was said about it.
  Requires the original words.

  LEVEL 2 — Working understanding:
  Can explain it in own words.
  Still requires paragraphs.

  LEVEL 3 — Deep understanding:
  Can explain it in one paragraph.
  Structure visible through language.

  LEVEL 4 — Structural understanding:
  Can express it in one sentence.
  Language nearly optional.

  LEVEL 5 — Invariant found:
  Can express it in one equation.
  Or fewer symbols.
  Language completely optional.
  The structure speaks for itself.

  ─────────────────────────────────────────────────────────────

  EXAMPLES OF LEVEL 5:

  E = mc²              →  mass and energy are the same thing
  F = ma               →  motion is governed by force and mass
  C ⊥ L                →  structure and language do not interact
  dI/dt > 0            →  this system is thinking
  Reach(s₀) ∩ Ω = ∅   →  harm is geometrically unreachable

  Each of these took years to find.
  Seconds to state.
  The time was in finding the invariant.
  Not in writing the symbol.

  The symbol is just what the invariant
  looks like when you finally see it.
```
